# Git/GitHub 零基础入门教程

欢迎！这是一个从零开始的 Git/GitHub 教程。**你不需要任何编程基础**，只需要跟着教程一步一步操作。

## 🗓️ 学习计划（每天30-60分钟）

| 天数 | 课程 | 内容 |
|------|------|------|
| 第1天 | [Lesson 1：认识 Git 和 GitHub](./lesson-01.md) | 概念、安装、配置 |
| 第2天 | [Lesson 2：你的第一次提交](./lesson-02.md) | 初始化仓库、add、commit |
| 第3天 | [Lesson 3：查看历史与撤销](./lesson-03.md) | log、diff、reset |
| 第4天 | [Lesson 4：连接 GitHub](./lesson-04.md) | 创建仓库、push、pull |
| 第5天 | [Lesson 5：分支](./lesson-05.md) | 创建合并分支 |
| 第6天 | [Lesson 6：协作与冲突](./lesson-06.md) | 多人协作、解决冲突 |
| 第7天 | [Lesson 7：Pull Request 工作流](./lesson-07.md) | Fork、PR、Code Review |
| 第8天 | [Lesson 8：.gitignore 与最佳实践](./lesson-08.md) | 忽略文件、提交规范 |
| 第9天 | [Lesson 9：实战项目](./lesson-09.md) | 综合练习 |
| 第10天 | [Lesson 10：常用技巧](./lesson-10.md) | 别名、stash、rebase |

## 🎯 学习方式

1. **每天一课**，按顺序学习，不要跳过
2. **边看边做**——教程里的每个命令都在你的电脑上实际操作
3. **遇到问题很正常**——试着用搜索引擎找答案，这也是程序员的核心技能
4. **完成练习**——每课末尾都有小练习，做完了再进入下一课
